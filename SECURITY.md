# Security Policy

This repository is an **independent fork** of SpaceXAI’s open-sourced Grok Build
CLI/TUI. It is **not** a security product and does not claim to be safe for
untrusted workloads.

## Reporting issues in *this* fork

For vulnerabilities introduced or worsened by fork-specific changes, contact the
repository maintainer privately (e.g. GitHub security advisories on this repo, or
a private channel the maintainer documents). Do not file a public issue with
exploit details.

## Upstream / official product

For the official SpaceXAI Grok Build product and its published source, use
SpaceXAI’s vulnerability process:

https://hackerone.com/x

Do not open public GitHub issues for security reports against the official
product either.
