# Security Policy

This repository is an **independent personal fork** of SpaceXAI’s open-sourced
Grok Build CLI/TUI. It is **not affiliated with xAI / SpaceXAI**, is **not** a
security product, and does not claim to be safe for untrusted workloads.

## Reporting issues in *this* fork

For vulnerabilities introduced or worsened by **fork-specific** changes, contact
the repository maintainer privately (e.g. GitHub Security Advisories on this
repo). Do not file a public issue with exploit details.

The maintainer of this fork is **not** xAI / SpaceXAI.

## Official SpaceXAI product / upstream source

For vulnerabilities in the **official** Grok Build product or SpaceXAI’s own
published tree, use their process:

https://hackerone.com/x

Do not open public GitHub issues for those reports either. Only use that channel
for issues that apply to **their** code/product — not for “I built this fork and
broke something locally” unless it also reproduces upstream.
